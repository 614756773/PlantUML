###流程图1：考试及成绩管理流程
####PlantUML源码如下：
```
@startuml
start
:安排考试;
:考试安排表;
:出卷;
 fork
   :A,B试卷;
 fork again
   :打印审批表;
   :审批签字;
   :打印审批表;
 end fork
 :打印试卷;
 :试卷;
 :参加考试;
 :答卷;
 :阅卷出成绩;
 fork
    :成绩单;
    if( 有不及格？) then( 有 )
        :安排补考;
     endif
     :补考安排表;
 fork again
    :答卷;
    :装订存档;
  end fork;
  :期末流程结束;
  stop
@enduml
```

###流程图2：客户维修服务流程
```
@startuml
start
:申请服务;
if (是新客户吗?) then (是)
  :登记客户信息;
else (不是)
endif
 :上门勘察;
:制定方案;
if (满意吗?) then (是)
  :签订服务合同;
else (否)
  stop
 endif
fork
  :安排工人;
fork again
  :安排材料;
endfork
:填写派工单;
:领取材料;
:上门服务;
:验收并填写反馈意见;
:交回派工单;
:结束收款;
stop
@enduml
```
